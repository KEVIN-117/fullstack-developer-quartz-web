# MOC Semanal

Esta sección se orienta a la organización temporal del grupo de estudio. Enfocada en notas semanales para reuniones, TODOs, reflexiones y progreso. Usa plantillas para consistencia y tracking a lo largo del tiempo.

## Propósito Principal
- Registrar agendas y outcomes de sesiones grupales.
- Asignar tareas y revisar avances fullstack.
- Reflexionar sobre lo aprendido para retención.

## Subsecciones Sugeridas
- [[Semana 1]]: Agenda, notas, TODOs (usa {{date}} para auto-fecha).
- [[Semana 2]]: Similar, con enlaces a proyectos o recursos.
- [[Plantillas]]: Modelo para nuevas semanas (agenda/TODOs/reflexiones).
- [[Progreso General]]: Resumen mensual o quarterly.
- [[Reuniones Pasadas]]: Archivos por fecha.

Usa tags como #semanal. Integra con plugins como Calendar para creación automática.
