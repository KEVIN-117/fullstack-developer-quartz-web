# MOC Bases de Datos

Esta sección se orienta al almacenamiento y manejo de datos en aplicaciones fullstack. Enfocada en modelos relacionales y no relacionales, queries y optimización. Útil para notas sobre persistencia, migraciones y escalabilidad de datos.

## Propósito Principal

- Aprender diseño de esquemas y consultas eficientes.
- Conectar con backend para CRUD operations.
- Comparar pros/contras de SQL vs NoSQL en contextos fullstack.

## Subsecciones Sugeridas

- [[SQL Básico]]: Queries, joins, índices en PostgreSQL/MySQL.
- [[NoSQL]]: Documentos en MongoDB, key-value en Redis.
- [[ORMs]]: Sequelize, Mongoose para abstracción de DB.
- [[Migraciones y Backups]]: Herramientas como Knex o MongoDB Compass.
- [[Optimización]]: Indexing, caching, sharding.
- [[Ejemplos y Snippets]]: Queries y schemas para apps reales.
