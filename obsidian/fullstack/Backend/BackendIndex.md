# MOC Backend

![[../assets/hero.png]]

Esta sección se orienta al desarrollo del lado del servidor en aplicaciones fullstack. Enfocada en lógica de negocio, manejo de datos, APIs y seguridad. Perfecta para notas sobre servidores, rutas y escalabilidad.

## Propósito Principal

- Documentar servidores, endpoints y autenticación.
- Integrar con frontend y bases de datos para flujos completos.
- Explorar patrones como MVC, microservicios y deployment.

## Subsecciones Sugeridas

- [[Node.js Básico]]: Runtime, módulos y async programming.
- [[Frameworks]]: Express, NestJS – rutas, middleware y errores.
- [[APIs]]: RESTful, GraphQL, WebSockets para real-time.
- [[Autenticación y Seguridad]]: JWT, OAuth, hashing y CORS.
- [[Herramientas]]: Docker, PM2, logging con Winston.
- [[Ejemplos y Snippets]]: Códigos para servers y endpoints.
