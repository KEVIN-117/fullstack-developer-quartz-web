# MOC Frontend

Esta sección se orienta a todo lo relacionado con el desarrollo del lado del cliente en aplicaciones fullstack. Enfocada en tecnologías web frontales, diseño responsivo y frameworks para interfaces de usuario interactivas. Ideal para notas sobre código visual, UX/UI y optimización de rendimiento.

## Propósito Principal

- Aprender y documentar conceptos básicos y avanzados de frontend.
- Conectar con backend para integración fullstack (e.g., APIs).
- Reflexionar sobre mejores prácticas para accesibilidad y performance.

## Subsecciones Sugeridas

- [[HTML y CSS Básico]]: Fundamentos de estructura y estilos.
- [[JavaScript Avanzado]]: ES6+, async/await, DOM manipulation.
- [[Frameworks]]: React, Vue o Angular – componentes, estados y hooks.
- [[Responsive Design]]: Media queries, flexbox, grid y mobile-first.
- [[Herramientas]]: Webpack, Babel, testing con Jest.
- [[Ejemplos y Snippets]]: Códigos reutilizables para UI components.
